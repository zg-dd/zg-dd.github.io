1. Overview

Go data types are split into basic types and composite types.

Basic types include integers, floating-point numbers, booleans, and strings. Composite types include arrays, slices, structs, functions, maps, channels, interfaces, etc.

2.2 Floating-point numbers

Go provides float32 and float64 following IEEE 754. On 32-bit systems the default is float32; on 64-bit systems it is float64.

  • float32: range of -3.4e38 to 3.4e38, uses 4 bytes
  • float64: range of -1.8e308 to 1.8e308, uses 8 bytes
var num1 float32 = 1.06
num2 := 3.1415926
fmt.Printf("Exact: %v; default 6 decimals: %f; fixed 3 decimals: %0.3f \n", num1, num2, num2)
var (
    num3 float32 = 3.1415926
    num4 float64 = 1.113
    num5         = 3.14e2
    num6         = 3.14e-2
)
fmt.Println(num3, num4, num5, num6)

Output:

Exact: 1.06; default 6 decimals: 3.141593; fixed 3 decimals: 3.142 
3.1415925 1.113 314 0.0314

2.3 Booleans

bool only has two values: true and false. Uninitialized booleans default to false.

var (
    bool1 bool
    bool2 = true
)
fmt.Println(bool1, bool2)

Output:

false true

2.4 Characters

Characters are single values and are represented as ints. Go exposes them via byte (ASCII) and rune (UTF-8).

  • byte: an alias for int representing ASCII values
  • rune: an alias for int representing UTF-8 code points

Note: UTF-8 is the universal encoding that extends ASCII to cover characters from multiple languages.

var a = 'A'
fmt.Printf("ASCII: %v; Char: %c \n", a, a)

In strings, ASCII letters occupy one byte and Chinese characters occupy three bytes. Use len to get the byte length.

2.4.2 Escape characters

Escape sequences start with \. Common ones include:

  1. \r: carriage return (moves to the beginning of the line)
  2. \n: newline (moves to the next line)
  3. \t: tab (helps align output)
  4. \': single quote
  5. \": double quote
  6. \\: backslash
var val1 string = "hello\r"
var val2 string = "world\n"
var val3 string = "hello world\t"
var val4 string = "'hello' \\ \"world\""
fmt.Println(val1, val2, val3, val4)

Output:

 world
 hello world     'hello' \ "world"

3. Type Conversion

Numeric conversion goes both ways: low-precision to high-precision and vice versa. Warning: converting from high to low precision can drop data because of differences in range.

3.1 Integer-to-integer conversions

var a int8 = 127
b := int64(a)
fmt.Printf("a=%v, type: %T\tb=%v, type: %T \n", a, a, b, b)
var c int64 = 127
d := int16(c)
fmt.Printf("c=%v, type: %T\td=%v, type: %T \n", c, c, d, d)

Output:

a=127, type: int8    b=127, type: int64 
c=127, type: int64   d=127, type: int16

3.2 Floating-point and integer conversions

var e int16 = 256
f := float32(e)
fmt.Printf("e=%v, type: %T\tf=%v, type: %T \n", e, e, f, f)
var g float64 = 3.145
h := int16(g)
fmt.Printf("g=%v, type: %T\th=%v, type: %T \n", g, g, h, h)

Output:

e=256, type: int16   f=256, type: float32 
g=3.145, type: float64   h=3, type: int16

3.3 Other type to string conversion via fmt

var (
    intV   = 123
    floatV = 3.145
    boolV  = true
    charV  = 'A'
)
str1 := fmt.Sprintf("%d", intV)
str2 := fmt.Sprintf("%f", floatV)
str3 := fmt.Sprintf("%t", boolV)
str4 := fmt.Sprintf("%c", charV)
fmt.Println(str1, str2, str3, str4)

Output:

123 3.145000 true A

3.4 strconv helpers

3.4.1 strconv.Atoi and strconv.Itoa

Tip: in Itoa the leading i stands for int -> string. In Atoi the trailing i stands for string -> int.

var str = "123"
num1, _ := strconv.Atoi(str)
fmt.Printf("value: %v\ttype: %T\n", num1, num1)
var num2 int64 = 456
str2 := strconv.Itoa(int(num2))
fmt.Printf("value: %v\ttype: %T\n", str2, str2)

Output:

value: 123     type: int
value: 456     type: string

3.4.2 strconv.ParseX functions

strconv.Parse performs conversions only if the source is valid.

var (
    str1 = "256"
    str2 = "3.145"
    str3 = "true"
)
num1, _ := strconv.ParseInt(str1, 10, 0)
fmt.Printf("value: %v\ttype: %T\n", num1, num1)
num2, _ := strconv.ParseFloat(str2, 32)
fmt.Printf("value: %0.3f\ttype: %T\n", num2, num2)
boo, _ := strconv.ParseBool(str3)
fmt.Printf("value: %v\ttype: %T\n", boo, boo)

Output:

value: 256     type: int64
value: 3.145   type: float64
value: true    type: bool

3.4.3 strconv.FormatX functions

var (
    intV   = 123
    floatV = 3.145e2
    booV   = true
)
str1 := strconv.FormatInt(int64(intV), 10)
fmt.Printf("value: %v\ttype: %T\n", str1, str1)
str2 := strconv.FormatFloat(floatV, 'f', -1, 64)
fmt.Printf("value: %v\ttype: %T\n", str2, str2)
str3 := strconv.FormatBool(booV)
fmt.Printf("value: %v\ttype: %T\n", str3, str3)

Output:

value: 123     type: string
value: 314.5   type: string
value: true    type: string

3.4.4 Other helpers

strconv also includes helpers like Append (to append to byte slices), IsPrint, IsGraphic, Quote, Unquote, and CanBackquote. Study them when needed; this reference is handy.
