1. Arrays

An array is a fixed-length collection of elements of the same type. Arrays are value types.

1.1 Declaring an array

var name = [length]<type>
var name = [length]<type>{}
name := [length]<type>{}

Example:

var arr1 [4]int
fmt.Printf("Type: %T; length: %v; data: %v\n", arr1, len(arr1), arr1)

Output:

Type: [4]int; length: 4; data: [0 0 0 0]

1.2 Initializing arrays

var arr1 [4]int
arr1[0] = 1
arr1[3] = 4
fmt.Printf("Method1: %T length: %v data: %v\n", arr1, len(arr1), arr1)
var arr2 = [3]float64{1, 2, 4.6}
fmt.Printf("Method2: %T length: %v data: %v\n", arr2, len(arr2), arr2)
arr3 := [3]string{"java", "Golang", "Python"}
fmt.Printf("Method3: %T length: %v data: %v\n", arr3, len(arr3), arr3)
arr4 := [...]int{2, 4, 6, 7, 8}
fmt.Printf("Method4: %T length: %v data: %v\n", arr4, len(arr4), arr4)
arr5 := [...]float64{0: 2, 2: 3.5, 5: 4.6, 7: 9}
fmt.Printf("Method5: %T length: %v data: %v\n", arr5, len(arr5), arr5)

Output:

Type: [4]int; length: 4; data: [0 0 0 0]
Method1: [4]int; length: 4; data: [1 0 0 4]
Method2: [3]float64; length: 3; data: [1 2 4.6]
Method3: [3]string; length: 3; data: [java Golang Python]
Method4: [5]int; length: 5; data: [2 4 6 7 8]
Method5: [8]float64; length: 8; data: [2 0 3.5 0 0 4.6 0 9]

1.3 Looping through arrays

arr6 := [3]string{"java", "Golang", "Python"}
for i := 0; i < len(arr6); i++ {
    fmt.Printf("idx%v: %v\t", i, arr6[i])
}
fmt.Println()
for index, value := range arr6 {
    fmt.Printf("idx%v: %v\t", index, value)
}

Output:

fori loop idx0: java; idx1: Golang; idx2: Python;
for range loop idx0: java; idx1: Golang; idx2: Python;

1.4 Multidimensional arrays

arr7 := [3][3]int{
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9},
}
fmt.Printf("Type: %T; length: %v; data: %v\n", arr7, len(arr7), arr7)
arr8 := [...][2]string{{"a", "b"}, {"c", "d"}}
fmt.Printf("Type: %T; length: %v; data: %v\n", arr8, len(arr8), arr8)
for i := 0; i < len(arr7); i++ {
    for j := 0; j < len(arr7[i]); j++ {
        fmt.Printf("row%v col%v: %v\t", i+1, j+1, arr7[i][j])
    }
    fmt.Println()
}
for row, rowValue := range arr8 {
    for col, value := range rowValue {
        fmt.Printf("row%v col%v: %v\t", row+1, col+1, value)
    }
    fmt.Println()
}

Output:

Type: [3][3]int; length: 3; data: [[1 2 3] [4 5 6] [7 8 9]]
Type: [2][2]string; length: 2; data: [[a b] [c d]]
row1 col1: 1; row1 col2: 2; row1 col3: 3;
row2 col1: 4; row2 col2: 5; row2 col3: 6;
row3 col1: 7; row3 col2: 8; row3 col3: 9;
row1 col1: a; row1 col2: b;
row2 col1: c; row2 col2: d;

2. Slices

A slice is a wrapper over an array and represents a dynamic-length view. Slices are reference types.

2.1 Declaring and initializing slices

var arr1 []int
fmt.Printf("Type: %T; len: %v; data: %v\n", arr1, len(arr1), arr1)
var arr2 = []float64{1, 2, 4.6}
fmt.Printf("Method1: %T; len: %v; data: %v\n", arr2, len(arr2), arr2)
arr3 := []string{"java", "Golang", "Python"}
fmt.Printf("Method2: %T; len: %v; data: %v\n", arr3, len(arr3), arr3)
arr4 := []int{2, 4, 6, 7, 8}
fmt.Printf("Method3: %T; len: %v; data: %v\n", arr4, len(arr4), arr4)
arr5 := []float64{0: 2, 2: 3.5, 5: 4.6, 7: 9}
fmt.Printf("Method4: %T; len: %v; data: %v\n", arr5, len(arr5), arr5)

Output:

Type: []int; len: 0; data: []
Method1: []float64; len: 3; data: [1 2 4.6]
Method2: []string; len: 3; data: [java Golang Python]
Method3: []int; len: 5; data: [2 4 6 7 8]
Method4: []float64; len: 8; data: [2 0 3.5 0 0 4.6 0 9]

2.2 Creating slices from arrays

arr1 := [5]int{10, 29, 49, 58, 32}
fmt.Printf("Type: %T; length: %v; data: %v\n", arr1, len(arr1), arr1)
s1 := arr1[:]
s2 := arr1[2:]
s3 := arr1[:2]
fmt.Printf("Full slice: %T; len: %v; data: %v\n", s1, len(s1), s1)
fmt.Printf("From idx2: %T; len: %v; data: %v\n", s2, len(s2), s2)
fmt.Printf("Before idx2: %T; len: %v; data: %v\n", s3, len(s3), s3)

Output:

Type: [5]int; length: 5; data: [10 29 49 58 32]
Full slice: []int; len: 5; data: [10 29 49 58 32]
From idx2: []int; len: 3; data: [49 58 32]
Before idx2: []int; len: 2; data: [10 29]

Summary: arr[start:end] extracts elements from index start (inclusive) to end (exclusive). Omit start or end to mean the beginning or end of the array.

2.3 Slice length and capacity

A slice has its own length and capacity: len returns the number of elements; cap counts how many elements remain from the first element to the end of the underlying array.

a := []int{2, 3, 4, 5, 6, 7}
fmt.Printf("Type: %T, len: %v, cap: %v, data: %v\n", a, len(a), cap(a), a)
b := a[3:]
c := a[:3]
fmt.Printf("%T, len: %v, cap: %v, data: %v\n", b, len(b), cap(b), b)
fmt.Printf("%T, len: %v, cap: %v, data: %v\n", c, len(c), cap(c), c)

Output:

Type: []int, len: 6, cap: 6, data: [2 3 4 5 6 7]
Type: []int, len: 3, cap: 3, data: [5 6 7]
Type: []int, len: 3, cap: 6, data: [2 3 4]

4. Value vs. reference types

Value types store independent copies, so updating one variable does not affect another. Reference types share the same underlying data, so modifying one reflects in the other. Slices are typical reference types.

a1 := [...]int{2, 3, 4, 5}
a2 := []int{2, 3, 4, 5}
b1 := a1
b2 := a2
b1[0] = 6
b2[0] = 6
fmt.Printf("a1=%v,b1=%v\n", a1, b1)
fmt.Printf("a2=%v,b2=%v\n", a2, b2)

Output:

a1=[2 3 4 5],b1=[6 3 4 5]
a2=[6 3 4 5],b2=[6 3 4 5]

5. Common slice utilities

5.1 make

make([]int, 10, 11) allocates a slice with length 10 and capacity 11.

slice1 := make([]int, 10, 11)
slice1[2] = 6
slice1[9] = 20
fmt.Printf("Type: %T, len: %v, cap: %v, data: %v\n", slice1, len(slice1), cap(slice1), slice1)

Output:

Type: []int, len: 10, cap: 11, data: [0 0 6 0 0 0 0 0 0 20]

5.2 append

Use append(slice, elems...) to grow slices.

slice2 := make([]int, 0, 0)
slice2 = append(slice2, 12, 14, 24)
fmt.Printf("%T len: %v cap: %v data: %v\n", slice2, len(slice2), cap(slice2), slice2)
slice3 := []int{3, 4, 5}
slice4 := append(slice2, slice3...)
fmt.Printf("%T len: %v cap: %v data: %v\n", slice4, len(slice4), cap(slice4), slice4)

Output:

Type: []int, len: 3, cap: 3, data: [12 14 24]
Type: []int, len: 6, cap: 6, data: [12 14 24 3 4 5]

5.3 copy (deep clone)

copy(dst, src) copies src into dst. Use it to avoid sharing the same backing array.

slice2 := make([]int, 0, 0)
slice2 = append(slice2, 12, 14, 24)
slice3 := []int{3, 4, 5}
slice4 := append(slice2, slice3...)
slice4Copy := make([]int, len(slice4))
copy(slice4Copy, slice4)
slice4Copy[0] = 6666
fmt.Println(slice4, slice4Copy)

Output:

[12 14 24 3 4 5] [6666 14 24 3 4 5]

5.4 Deleting from a slice

Go has no built-in delete. Combine slicing with append to drop a range.

s1 := []int{1, 2, 3, 4, 5, 6}
delIdx := 3
s1 = append(s1[:delIdx], s1[delIdx+1:]...)
fmt.Println(s1)

Output:

[1 2 3 5 6]

5.5 Sorting slices

5.5.1 Ascending

Use sort.Ints, sort.Float64s, or sort.Strings.

intList := []int{3, 4, 32, 32, 423, 12}
floatList := []float64{3.13, 4.2, 5.3, 1.1}
stringList := []string{"D", "S", "A", "C"}
sort.Ints(intList)
sort.Float64s(floatList)
sort.Strings(stringList)
fmt.Println(intList)
fmt.Println(floatList)
fmt.Println(stringList)

Output:

[3 4 12 32 32 423]
[1.1 3.13 4.2 5.3]
[A C D S]

5.5.2 Descending

Combine sort.XXXSlice, sort.Reverse, and sort.Sort.

intList := []int{3, 4, 32, 32, 423, 12}
floatList := []float64{3.13, 4.2, 5.3, 1.1}
stringList := []string{"D", "S", "A", "C"}
sort.Sort(sort.Reverse(sort.IntSlice(intList)))
sort.Sort(sort.Reverse(sort.Float64Slice(floatList)))
sort.Sort(sort.Reverse(sort.StringSlice(stringList)))
fmt.Println(intList)
fmt.Println(floatList)
fmt.Println(stringList)

Output:

[423 32 32 12 4 3]
[5.3 4.2 3.13 1.1]
[S D C A]

6. Tips and tricks

6.1 Editor hints

When you type . after a variable the editor shows suggestions. Items ending with ! insert scaffolding code.

HintMeaning
append!Append slice items
copy!Copy slice
for!standard for loop
forr!for range loop
ifnotnil!if var != nil guard
last!shorthand for a[len(a)-1]
len!shorthand for len(a)
print!print variable
range!alias for forr!
reverse!reverse slice
sort!custom sort scaffolding
var!assign to a new variable

6.2 Inspecting signatures

  1. Hover for over 1s to view parameter and return metadata.
  2. Ctrl+Click enters the definition with full documentation. Use translation tools if needed.
