1. Closures

1.1 What is a closure?

A closure is a function combined with the environment that it captures. In Go, closures can access and modify variables defined in their enclosing scopes even after the outer function returns.

1.2 Closure properties

  1. A closure can access variables from its outer scope.
  2. A closure can modify those outer-scope variables.
  3. The variables persist after the outer function exits.

1.3 Closure example

package main
import "fmt"
func adder() func(int) int {
    sum := 0
    return func(x int) int {
        sum += x
        return sum
    }
}
func main() {
    pos, neg := adder(), adder()
    for i := 0; i < 10; i++ {
        fmt.Println(pos(i), neg(-2*i))
    }
}

Each call to adder returns a distinct closure that captures its own sum variable. The output shows two independent sums growing separately.

2. Types

2.1 Custom types

Go lets you define new type names with the type keyword.

2.2 Type example

package main
import "fmt"
type addFun func(int) int
type selfint int
func adder2(x int) addFun {
    return func(y int) int {
        return x + y
    }
}
func main() {
    var eg addFun = adder2(10)
    fmt.Println(eg(15))
    var num1 selfint = 10
    var num2 int = 20
    fmt.Println(num1 + selfint(num2))
    fmt.Println(int(num1) + num2)
}

Output: 25, 30, 30. selfint cannot be mixed with int without conversion.

3. Defer

3.1 How defer works

A defer statement postpones a function call until the surrounding function returns.

3.2 Defer example

package main
import "fmt"
func main() {
    defer fmt.Println("world")
    fmt.Println("hello")
}

Output: hello followed by world because the deferred call runs after main finishes.

4. Panic and Recover

panic is similar to throwing an exception; recover catches it. Go does not have try..catch syntax.

4.1 Usage

  • panic signals an unrecoverable error.
  • recover captures the panic so the program can continue.

4.2 Example

package main
import "fmt"
func main() {
    defer func() {
        if r := recover(); r != nil {
            fmt.Println("Recovered from", r)
        }
    }()
    fmt.Println("Starting the program")
    panic("Something went wrong!")
    fmt.Println("This line will not run")
}

Output: Starting the program and then Recovered from Something went wrong! because the deferred function recovered from the panic.

5. The time package

The time package provides utilities for working with dates and times, including formatting, parsing, duration math, and comparisons.

currentTime := time.Now()
fmt.Println("Current Time:", currentTime)
year := currentTime.Year()
month := currentTime.Month()
day := currentTime.Day()
hour := currentTime.Hour()
minute := currentTime.Minute()
second := currentTime.Second()
fmt.Printf("Current Time: %d-%02d-%02d %02d:%02d:%02d\n", year, month, day, hour, minute, second)
fmt.Println("Formatted:", currentTime.Format("2006-01-02 15:04:05"))
timeStr := "2023-01-01 12:00:00"
parsedTime, err := time.Parse("2006-01-02 15:04:05", timeStr)
if err != nil {
    fmt.Println("Failed to parse:", err)
    return
}
fmt.Println("Parsed:", parsedTime)
duration := time.Duration(10) * time.Second
fmt.Println("Duration:", duration)
newTime := currentTime.Add(duration)
fmt.Println("New Time:", newTime)
if currentTime.Before(parsedTime) {
    fmt.Println("Current time is before parsed time")
} else if currentTime.After(parsedTime) {
    fmt.Println("Current time is after parsed time")
} else {
    fmt.Println("Current time equals parsed time")
}
if currentTime.Equal(parsedTime) {
    fmt.Println("Current time equals parsed time")
}

Summary:

  1. time.Now()
  2. Year/Month/Day/Hour/Minute/Second
  3. currentTime.Format("2006-01-02 15:04:05")
  4. time.Parse("2006-01-02 15:04:05", timeStr)
  5. time.Duration(10) * time.Second
  6. currentTime.Add(duration)
  7. currentTime.Before/After/Equal(parsedTime)
